# Criar projetos Flutter fora do Android Studio para evitar erros com Windows Defender

## 0. Verificando requisitos:

```bash
flutter doctor
```

### 1. Criando o projeto:

```bash
flutter create --platforms=android,web my_project
```
### 2. Entrando na pasta do projeto:

```bash
cd my_project
```
### 3. Rodando o projeto:
```bash
flutter run
```
